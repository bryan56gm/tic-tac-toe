# tic-tac-toe

Juego de tres en raya para 2 jugadores locales en el mismo dispositivo.

## Setup

```bash
pnpm install
pnpm dev
```

## Evolución del proyecto

| Necesidad | Acción |
|---|---|
| Nuevo feature | `/hs-feature {nombre}` |
| Nueva decisión técnica | `docs/decisions/{NNNN}-{titulo}.md` |
| Nuevo skill | `.claude/skills/{nombre}/SKILL.md` |
| Regla por módulo | `.claude/rules/{modulo}.md` con `paths:` |

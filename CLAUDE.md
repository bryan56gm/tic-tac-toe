# tic-tac-toe

Juego de tres en raya para 2 jugadores locales en el mismo dispositivo.

## Stack
Vite + React 18 + TypeScript strict + Tailwind CSS v4. Sin backend ni auth.

## Notas
- Toda la lógica del juego vive en `src/features/game/useGame.ts`
- Sin persistencia: el estado se reinicia al recargar
